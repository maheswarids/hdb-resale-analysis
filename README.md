## HDB Resale Analysis (2015–2016)

## Project Overview

This project analyzes HDB resale flat prices in Singapore to uncover trends based on location, flat type, and time.

The goal is to transform raw data into meaningful insights using data analysis and visualization techniques.

---

## Objectives

* Understand price trends over time (2015–2016)
* Identify which towns have higher resale prices
* Analyze how flat types impact pricing
* Build an interactive dashboard for better decision-making

---

## Tools & Technologies

* Python (Pandas, NumPy)
* Data Cleaning & Feature Engineering
* Power BI (Dashboard & Visualization)
* Google Colab

---

## Data Cleaning Steps

* Check for missing values
* Converted data types (especially date columns)
* Ensured consistency in categorical data

---

## Feature Engineering

Created new features from existing data:

* Extracted **Year** from date
* Extracted **Month Number**
* Extracted **Month Name**

These features helped in time-based analysis.

---

## Key Insights

* Resale prices increased from 2015 to 2016
* Towns like Bukit Timah and Central Area have higher average prices
* Larger flats (Executive, Multi-Generation) are more expensive
* Monthly trends show fluctuations but overall stable growth

---

## Dashboard

The Power BI dashboard includes:

* Average resale price by town
* Average resale price by flat type
* Year-wise trend
* Month-wise trend
* Interactive slicers (Town & Flat Type)
